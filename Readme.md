## Installation

### MSYS2

```
yes | pacman -S git
git clone git@github.com:smack0007/.dotfiles.git ~/.dotfiles
```

### WSL 

```
wsl --update
wsl --shutdown
```

### WSL Debian

```
wsl --install Debian
sudo apt install git
git clone git@github.com:smack0007/.dotfiles.git ~/.dotfiles
```

### Common

```
sh ./install.sh
```
